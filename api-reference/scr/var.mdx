# Referência de Variáveis

Esta seção descreve as principais variáveis utilizadas nas respostas da API, explicando seus significados e funções.  
Cada campo representa um elemento relevante para a análise de crédito, como:

- Identificação do cliente
- Composição da carteira
- Faixas de vencimento (vCOD)
- Limites contratados
- Modalidades de crédito
- Responsabilidades e riscos

O objetivo é fornecer uma visão clara e estruturada dos dados retornados, auxiliando analistas, integradores e sistemas na correta interpretação e uso dessas informações para tomada de decisão.
