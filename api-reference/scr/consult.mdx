<RequestExample>

```bash cURL
curl -X POST https://sandbox-api.databox360.com.br/api/scr \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
        "documento": "12345678901",
        "mes": "03",
        "ano": "2025"
      }'
```

```javascript JavaScript
const response = await fetch("https://sandbox-api.databox360.com.br/api/scr", {
  method: "POST",
  headers: {
    Authorization: "Bearer YOUR_JWT_TOKEN",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    documento: "12345678901",
    mes: "03",
    ano: "2025",
  }),
});

const scrData = await response.json();
console.log(scrData);
```


```python Python
import requests

url = "https://sandbox-api.databox360.com.br/api/scr"
headers = {
    "Authorization": "Bearer YOUR_JWT_TOKEN",
    "Content-Type": "application/json"
}
payload = {
    "documento": "12345678901",
    "mes": "03",
    "ano": "2025"
}

response = requests.post(url, json=payload, headers=headers)
scr_data = response.json()
```


```php PHP
<?php
$url = 'https://sandbox-api.databox360.com.br/api/scr';
$data = [
    'documento' => '12345678901',
    'mes' => '03',
    'ano' => '2025'
];

$options = [
    'http' => [
        'header' => [
            "Authorization: Bearer YOUR_JWT_TOKEN",
            "Content-type: application/json"
        ],
        'method' => 'POST',
        'content' => json_encode($data)
    ]
];

$context = stream_context_create($options);
$result = file_get_contents($url, false, $context);
$scr_data = json_decode($result, true);
?>
```
