---
title: "Modalidades "
description: "As modalidades de operação representam a classificação detalhada das operações de crédito registradas no SCR Bacen. Cada operação é identificada por um código único (modalidade) e está vinculada a um grupo de modalidades mais amplo, conforme estrutura definida pelo Banco Central."
---

A estrutura é composta por:

- **Categorias:** representam agrupamentos amplos de operações com características similares, como _Empréstimos_, _Financiamentos_, _Direitos Creditórios_, entre outras. 
- **Modalidades:** são operações específicas dentro de cada categoria, que definem com maior precisão o tipo de contrato informado ao SCR. Exemplos incluem: _Crédito Pessoal Consignado_, _Crédito Rotativo_, _Vendor_, _Financiamento de Veículos_, entre outros.

Cada código `Modalidade` deve ser interpretado de forma consistente com sua descrição oficial e utilizado conforme a natureza da operação e o contexto do relatório, conforme detalhado a seguir:
