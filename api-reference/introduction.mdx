A API Databox360 foi desenvolvida de acordo com os melhores padrões REST.

As operações são realizadas por meio de requisições aos endpoints, utilizando os verbos HTTP apropriados e corpos de mensagem em JSON.

As respostas podem ser interpretadas através do código de status HTTP, permitindo que qualquer aplicação consuma a API de forma simples, padronizada e independente da linguagem utilizada.

Para facilitar ainda mais o processo de integração, disponibilizamos uma chave de testes para uso em nosso ambiente Sandbox. Assim, você pode realizar quantos testes forem necessários, sem qualquer cobrança das transações requisitadas.

👉 Sandbox: https://sandbox-api.databox360.com.br/api

📦 Collection Postman: https://databox360.com.br/downloads/databox360-sandbox.postman_collection
